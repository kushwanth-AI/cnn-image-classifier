# CNN Image Classifier (PyTorch)

This project builds a Convolutional Neural Network using PyTorch to classify handwritten digits (MNIST dataset).

## Features
- CNN architecture
- Training & testing pipeline
- Accuracy evaluation

## Tech Stack
- PyTorch
- Google Colab

## 📊 Results
- Training Accuracy: ~98%
- Test Accuracy: ~97%

## 🧠 Concepts Used
- Convolution
- Pooling
- Flatten
- Fully Connected Layers
- Loss Function (CrossEntropy)
- Optimizer (Adam)
